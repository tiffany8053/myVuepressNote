# Vuepress 部署到 github

由於github page是放置靜態資源,須先將專案文件打包成靜態資源

步驟：

  1. 將文件打包
yarn docs:build
1

會在docs/.vuepress資料夾下多一個 dist文件夾

  1. 到git 創建新的 Repository

git Repository專案名稱 與本機專案名稱同名

注意：

Repository 一定要公開 public 才能使用github page

  1. 到dist資料夾加入git管理
    cd docs/.vuepress/dist
    git init
    git add -A (-A：表示全部加入追蹤)
    git commit -m '輸入描述'
1
2
3
4
  1. 將本機專案部署連至github
    git remote add origin https://github.com/tiffany8053/0111test.git
    git branch -M main
    git push -u origin main
1
2
3
  1. 在到GitHub -> setting -> GitHub Pages none 改為 main,並save保存
