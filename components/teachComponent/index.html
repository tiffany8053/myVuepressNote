    # 在vuepress md文件中使用vue組件

    將vue組件放置.vuepress/components目錄下

    在需要使用vue組件的.md文件中輸入<ClientOnly><目錄-自定義組件/></ClientOnly>就可使用。

    <ClientOnly>
  <button-TheButton/> // 自定義組件
</ClientOnly>
