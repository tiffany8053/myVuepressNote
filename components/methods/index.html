# 強調

使用星號（ * ）和底線（_）作為標記強調字詞的符號，被* 或_包圍的字詞會被轉成用 < em > 標籤包圍，用兩個*或_包起來的話，則會被轉成< strong >

範例：

＊test＊
＿使用下底線＿
＊＊test＊＊
＿＿使用兩個下底線＿＿

# 程式碼

如果要標記一小段行內程式碼，你可以用反引號把它包起來（`)

即： ` `

範例：

use <script></script> to write code.

若要在畫面顯示反引號,請用2個反引號做開頭與結尾,中間個放空格後,再放反引號,
即：` ` + 空格 + <p> + 空格 + ` `

範例：

today is `rainy` day.
